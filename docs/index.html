Install in seconds

Install dropclass locally with a Python virtual environment and run the test suite.

python3 -m venv .venv
source .venv/bin/activate
pip install -e '.[dev]'
pytest

How it works

A four-stage engine: discovery, enrollment, formation, and verified talent export.

dropclass/
├── discovery/   Class Central search + catalog normalization
├── coursera/    Platform student loop (future)
├── formation/   Enrollment, memory, and talent store
└── api/         FastAPI routes mounted by Atelier
